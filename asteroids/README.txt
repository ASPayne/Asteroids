Welcome to Asteroids!

This game was inspired by the classic Atari game: ASTROIDS

Created for a school class, programmer Andrew Payne designed
and developed this game in C++ to meet and exceed standard
requirements for recreating the game.

Controls:

UP...........................................Apply Thrusters
LEFT......................................Turn The Ship Left
RIGHT....................................Turn The Ship Right
ESC.......................................Pause/Unpause Game
SPACE BAR......................................Fire A Bullet

This program has default values. however, by modifying and
recompiling the "asteroidVariables.h" the user can make the
game more enjoyable

Easter Eggs:

A.................................Turn ON/OFF Android Robots
S................................Turn ON/OFF StarWars X-Wing
